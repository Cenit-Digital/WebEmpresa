# UI kit — Web (Cénit Digital)

Recreación de la landing corporativa de Cénit Digital con los tokens y el
lenguaje visual del sistema. Sirve como **starting point** ("plantilla") y como
vista de referencia.

- `index.html` — landing: cabecera (logo + nav + toggle de tema + "Hablamos"),
  hero (eyebrow, titular, subtítulo, 2 CTA, 4 estadísticas), sección de paquetes
  (3 tarjetas, la central destacada) y pie.
- El toggle de la cabecera conmuta `data-theme` en `<html>` (claro ⇄ oscuro).

La **home completa** (servicios en zigzag, sectores, contacto con formulario)
está montada en `Cenit Home (referencia) — standalone.html` (raíz del proyecto).
Amplía este kit desde ahí si necesitas todas las secciones.
