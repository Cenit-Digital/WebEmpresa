# Tag

Etiqueta de categoría (pill) para las tarjetas de servicio y similares.

```jsx
<Tag>Marketing</Tag>
```

- Texto corto en versalitas (se muestra en mayúsculas por CSS).
- Colores desde tokens: tinta `--color-tag-ink` sobre fondo `--color-tag-bg`.
- Decorativa: no es interactiva. Para acciones usa `Button`.
